Optional Features added:
Themed as Monkeys finding Bloons

1. Welcome Screen animation, Button to skip anmiation, and Autoskip after animation end
2. Sounds for Game (Scanning & Popping)

7. Has background music that pauses when leaving the app, and resumes at the same position when resuming
